# Informatik-Projekt
Programmidee: Aktiensimulation

## Funktionen

- Aktien mit verschiedenen Volatilitäten
- Auflisten der Aktien
- Kaufen und Verkaufen von Aktien
- Aktien verhalten sich dynamisch in jedem "Spiel-Tag"
- Interaktiv, Ansi-Ausgabe im Terminal
- Je nach Situation generierte Reaktionen (wie zum Beispiel `(ﾉಠдಠ)ﾉ︵┻━┻`)
